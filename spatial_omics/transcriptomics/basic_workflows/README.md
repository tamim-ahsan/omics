Within this folder, I store scripts as I explore the **workflows and packages** to analyze spatially resolved transcriptomic (SRT) datasets from various platforms.

The following explorations have been done so far -
| SRT Platforms | Packages | Links to the methods' papers |
|---------------|----------|-----------------|
| Visium HD | Sainsc | https://doi.org/10.1002/smtd.202401123 |
| Stereo-seq | Stereopy | https://doi.org/10.1038/s41467-025-58079-9 |
| Xenium | Seurat | https://doi.org/10.1038/s41587-023-01767-y |
